## 1. Propósito del repositorio

Este repositorio documenta una **ruta de estudio y proyectos reales** diseñada para
evolucionar de **Backend Developer PHP** a **Backend Engineer**, demostrando:

- Capacidad de diseño de sistemas
- Pensamiento arquitectónico
- Decisiones técnicas justificadas
- Código backend mantenible y escalable
- Dominio del negocio, no solo del framework

No es un curso ni un tutorial.
Es una **ruta de ingeniería aplicada**.

---

## 2. Perfil de partida (contexto real)

- +4 años de experiencia profesional
- PHP nativo y Laravel en producción
- APIs, microservicios y endpoints reales
- SQL relacional (PostgreSQL / MariaDB)
- Docker, Linux, GitLab
- Frontend como complemento (no core)

Este plan **no reinventa el perfil**, lo profundiza.

---

## 3. Objetivo final

Al finalizar esta ruta, el perfil esperado es:

> **Backend Engineer (PHP-focused, stack-agnostic)**

Capaz de:
- Diseñar sistemas backend completos
- Elegir arquitecturas conscientemente
- Explicar trade-offs técnicos
- Mantener y escalar sistemas productivos
- Adaptarse a otros stacks (Node/Nest) sin perder foco

---

## 4. Metodología de estudio

- **3 sesiones por semana**
